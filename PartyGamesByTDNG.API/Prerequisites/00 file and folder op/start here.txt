Initial Setup

1. Create directory "webservices" from the root folder
mkdir webservices

2. Go to webservices folder
cd webservices

3. Create directory werp
mkdir werp


To delete all files and subdirectories of werp folder
rm -r /webservices/werp/*

=============================================

Transfer files via SCP
on local computer

scp -r <directory or folder to copy>/. root@thedotnetguy.com:/webservices/werp

ex:
scp -r Release/. root@thedotnetguy.com:/webservices/werp
