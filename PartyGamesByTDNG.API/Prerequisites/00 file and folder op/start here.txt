Initial Setup

1. Create directory "webservices" from the root folder
mkdir webservices

2. Go to webservices folder
cd webservices

--------------------
If webservices folder already exist

3. Create directory partygamesbytdng
mkdir partygamesbytdng

rm -r /webservices/werp/*
